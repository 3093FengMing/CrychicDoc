# 启动脚本事件列表

## 前言

- 放在startup_script文件夹下生效。

## StartupEvents 启动事件

|   事件名称    |   用途    |   示例    |
|:------------:|:---------:|:---------:|
|   init   |   初始化时触发   |   -   |
|   modifyCreativeTab   |   修改创造物品栏时触发   |   -   |
|   postInit   |   初始化完成后触发   |   -   |
|   recipeSchemaRegistry   |   配方类型注册   |   -   |
|   registry   |   注册   |   -   |
